## Overview

A private permissioned blockchain network purpose-built for secure data brokerage, disaster recovery workflows, and tamper-proof transaction records. Leverages CometBFT consensus with a custom Rust ABCI application layer.

### Features

- CometBFT Byzantine fault-tolerant consensus
- Custom Rust ABCI application
- Permissioned validator network
- Secure transaction record storage
- Disaster recovery data brokerage
