## Overview

A comprehensive toolkit for analyzing and deploying tokens across multiple chains. Includes automated security scanning, contract compilation, and simulation capabilities.

### Features

- Smart contract security detectors
- Foundry wrapper for compilation
- Session-based simulation engine
- CLI and API server interfaces
