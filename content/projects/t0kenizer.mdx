## Overview

A full-stack tokenization platform for issuing and managing real-world asset (RWA) tokens across multiple EVM-compatible chains. Handles token lifecycle from creation through compliance and distribution.

### Features

- Multi-chain RWA token issuance
- Compliance and regulatory framework integration
- Admin dashboard for token management
- Smart contract deployment and interaction
